# 🔐 Password Strength Checker (Python + Tkinter GUI)

A modern, user-friendly Password Strength Checker built using Python and Tkinter. This application evaluates password security based on multiple criteria and provides real-time feedback along with actionable suggestions to help users create strong and secure passwords.

---

## 📖 Overview

Weak passwords are one of the most common causes of security breaches. This project aims to improve password security awareness by analyzing user input and guiding them toward creating stronger passwords.

The application features a clean graphical user interface (GUI) and includes usability enhancements like a show/hide password toggle and color-coded strength indicators.

---

## ✨ Key Features

### 🔍 Password Analysis
- Evaluates password strength using multiple security rules
- Checks for:
  - Minimum length (8+ characters)
  - Uppercase letters (A–Z)
  - Lowercase letters (a–z)
  - Numbers (0–9)
  - Special characters (!@#$%^&*)

### 🚦 Strength Classification
- ❌ Very Weak (common passwords)
- ❌ Weak
- ⚠️ Medium
- ✅ Strong

### 🎨 Visual Feedback
- Color-coded results:
  - Red → Weak
  - Orange → Medium
  - Green → Strong

### 💡 Smart Suggestions
- Provides real-time improvement tips
- Helps users understand how to strengthen their passwords

### 🔐 Security Usability
- Show/Hide password toggle 👁️
- Prevents input errors with alerts

### 🖥️ GUI Interface
- Built using Tkinter
- Dark-themed modern UI
- Clean layout for better user experience
